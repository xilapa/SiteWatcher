title Worker flow

participant PyScheduler
participant Worker API
participant Queue
participant WatcherService
participant NotificationService

PyScheduler->Worker API:Request to run a periodic watcher
activate Worker API
Worker API->Queue:Post job on the queue
Queue ->Worker API:Confirm job is received
Worker API->PyScheduler: Confirm job is created
deactivate Worker API
Queue-->>WatcherService: Get the job to be executed
activate WatcherService
WatcherService-->>Queue: Post notifications on the queue
Queue-->>NotificationService:Get the notification to be sent
activate NotificationService
NotificationService-->>Queue:Confirm the notification was sent
deactivate NotificationService
WatcherService-->>Queue: Confirm the job was done
deactivate WatcherService
